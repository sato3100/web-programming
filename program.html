GitHub
リモートリポジトリを提供している、webサービスです。オープンライセンスがあるソースコードは、「MIT」、「Apache 2.0」、「GPLv3」の3段階で利用ができます。
とほほのWWW入門
ウェブサイト制作の基礎的な知識を学べるサイトです。商用・私用に関わらず無断で利用・改造・修正・再配布可能です。条件を守れば文章も再配布可能です。
MDN Web Docs
ウェブ標準及びMozillaプロジェクトの開発文書のためのMozillaの公式ウェブサイトです。2010年8月20日以降に追加された例は、全てパブリックドメインになっています。変更された日付に注意が必要です。
Codejump
「模写コーディング」を通し、コーディング練習を行うサイトです。ソースコードについては、著作権フリーです。（有料教材のコンテンツは対象外です。）
